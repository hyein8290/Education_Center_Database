## 📝 Main Function
### 1. 관리자
* 기초 정보(과정, 과목, 강의실, 교재) 관리
* 교사 계정 관리
* 개설 과정 관리
* 개설 과목 관리
* 교육생 관리
* 시험 관리 및 성적 조회
* 출결 관리
* 상담 일지 관리
* 취업 정보 관리
